**Stop running your app like a solo developer. Run it like a company.**

## What's included

### 4 layers, 105 agents

| Layer | What it does |
|---|---|
| **Monitoring** | Watches your entire stack 24/7 — deploys, errors, database health, automations, APIs, OAuth, SSL, quotas. You know before it breaks. |
| **Healing** | Build errors auto-fixed. Broken workflows restarted. Stripe webhooks re-enabled. Database migrations safety-checked. Problems handled without you. |
| **Growth** | UX friction detection. Copy review. Churn analysis. Competitor monitoring. A/B experiment design. Clear actions, not reports. |
| **Learning** | Every fix is remembered. Mistakes don't repeat across projects. The system gets smarter every session. |

### Also included
- **6 lifecycle hooks** — auto-trigger agents on events (file edits, session start/stop)
- **5 slash commands** — `/improve`, `/start`, `/pr`, `/bug`, `/health`
- **6 GitHub Actions workflows** — auto-PR, auto-merge, bug review, security scanning, auto-labeling
- **Global settings** — pre-tuned for daily development work

## Quick install

**Full instructions in [SETUP.md](SETUP.md) — 30 minutes, no terminal experience required.**

Short version:
1. Download this repo as a ZIP
2. Copy `agents/`, `hooks/`, `skills/` into `~/.claude/`
3. Replace `YOUR-PROJECT-1` etc. with your actual project names
4. Open Claude Code and type `run session-commander`

---

## The 3 commands you'll use every day

```
run session-commander   ← start of day briefing
/improve                ← end of session, captures everything learned
/start                  ← mid-session health check
```
