# MEMORY INDEX

This file is auto-populated as you use the system.

The memory system builds up automatically — each session adds new patterns, learnings, and project-specific knowledge. You don't need to fill this in manually.

After your first few sessions with `/improve`, you'll start seeing entries appear here automatically.

## How it works

- **Session close** (`/improve`) → `session-learner` captures what was discovered and writes it here
- **Bug fixes** → `lesson-extractor` reads your commits and adds patterns
- **Agent runs** → specialist agents write findings to relevant memory files

## Getting started

Run this in Claude Code to initialize:

```
run session-commander
```

Your memory system will start building from there.
